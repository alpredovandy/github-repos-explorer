# Github Repositories Explorer

This project build using Stack:

-   Vite
-   React
-   Vitest
-   Testing Library
-   Tanstack React Query
-   TailwindCSS
-   daisyUI
-   Axios
-   Zod
-   Github API

## Scripts

-   How to Run `=>` `yarn dev`
-   How to Build `=>` `yarn build`
-   How to Test `=>` `yarn test`

## Demo

Link `=>` https://github-repos-explorer-nine.vercel.app
